DELL BIOS Unlock

Pre:

After confirming you have a DELL PC or motherboard, find its code which for DELL brand name computers can be found when
you are prompted for the password in the BIOS. Goto this website and enter the code to get the master recovery code.
You will want to take a picture or write down the resulting codes and for DELL brand computers the proper code will be
the one not labled with "(old)".

Hack:

Open your PC BIOS and when promped for a password, enter in the master recovery code
(Note: For some PCs you will need to CTRL+ENTER to submit)
To open the BIOS, during boot/startup you will have to press one of many possile buttons
including: F12, F2, or Escape
